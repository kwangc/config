# Model Class

High-level model families: **VLM**, **LLM**, **VLA**.

---

## Overview

- **VLM** — Vision-Language Model: image/video + text in, text (or structured output) out.
- **LLM** — Large Language Model: text in, text out; foundation for reasoning and instruction-following.
- **VLA** — Vision-Language-Action: image/video + (optional) language in, **robot actions** out; builds on VLM/LLM with an action head.
